What's new

Release notes and version history for OpenForge.

0.2.5 Latest March 28, 2026

Central pane tab system

  • Tabbed editor pane — open multiple files, diffs, and chats as tabs in the central panel
  • CodeMirror editor — syntax-highlighted file viewer with 15+ language grammars
  • Side-by-side diff viewer — CodeMirror merge view comparing worktree against base branch
  • Changes overview — summary view with file count, lines added/removed, and status badges
0.2.4 March 28, 2026

Workspace status & app icon

  • Workspace hover card — hover over any workspace to see task, status, lines changed, and last activity
  • Monochrome app icon — new minimal icon that fits macOS design language
0.2.3 March 28, 2026

File-type icons

  • File-type icons — distinct icons for JS, TS, Python, Rust, Go, and 20+ more file types across the file tree, diff viewer, and tabs
0.2.2 March 28, 2026

Fictional workspace names

  • 200+ fictional names — workspaces are now named after places from Tolkien, Harry Potter, Star Wars, anime, mythology, and more
0.2.1 March 28, 2026

Website downloads

  • Direct download links — separate Apple Silicon and Intel Mac download buttons on the website, auto-resolved from GitHub Releases
0.2.0 March 28, 2026

Build & release pipeline

  • Auto-release on merge — GitHub Actions workflow builds DMGs for both architectures and creates a release automatically
  • DMG build fix — added --bundles dmg flag to produce correct macOS disk images
  • README — comprehensive docs with architecture overview, screenshots, and build instructions
0.1.0 Initial release March 27, 2026

Core application

  • Three-panel layout — resizable sidebar, central chat pane, and right panel with files, diff, and terminal
  • Git worktree management — create, archive, and restore isolated worktrees per task
  • Claude Code integration — multi-turn chat sessions with Haiku, Sonnet, and Opus model selection
  • Streaming output — NDJSON parsing with tool call rendering, thinking blocks, and cost tracking
  • Checkpoint system — automatic git-ref snapshots before each message with one-click revert
  • Integrated terminal — xterm.js with full PTY support per workspace
  • Diff viewer — see changed files vs base branch with add/modify/delete/rename status
  • File tree — browse and view all files in the worktree
  • Repo scripts — configurable setup, run, and archive shell scripts per repository

Design & infrastructure

  • Monochrome theme — luxury dark and light modes with automatic system preference detection
  • GitHub Pages site — landing page with download links and project overview
  • Streaming fixes — fixed tool status updates and added requestAnimationFrame throttling for smooth rendering
